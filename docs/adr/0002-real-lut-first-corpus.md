# Real LUT First Corpus

The v1 dataset will prioritize real and expert-derived global LUTs over synthetic residual LUTs. The intended source order is PPR10K-derived expert LUTs, FiveK-derived expert LUTs, Fresh LUTs, G'MIC/RawTherapee HaldCLUTs, and smaller public LUT packs, while ISP/HDR/DPED-style camera-pipeline transforms are excluded to keep the corpus focused on photographic color grading.
