# Derived LUT Extraction

PPR10K-derived expert LUTs will be created first by applying expert XMP targets to an identity color grid, with pair fitting used as a fallback or validation path. FiveK-derived expert LUTs will be created by fitting a global LUT from each source image to its expert-retouched target, because FiveK is naturally represented as before/after expert image pairs rather than directly reusable edit presets.
